# Datasets

This folder does not contain the actual datasets because the dataset files are very large.

The datasets used for training the AI models can be downloaded from the following links.

---

# Skin Disease Dataset

Source:
https://www.kaggle.com/datasets/pacificrm/skindiseasedataset

Description:
This dataset contains multiple categories of skin diseases such as:
- Acne
- Psoriasis
- Vitiligo
- Eczema
- Warts
- Skin Cancer
- Rosacea
and more.

The dataset was used to train the skin disease classification model.

---

# Oral Disease Dataset

Source:
https://www.kaggle.com/datasets/salmansajid05/oral-diseases

Description:
This dataset contains oral disease image categories including:
- Gingivitis
- Caries
- Mouth Ulcer
- Tooth Discoloration

The dataset was used to train the oral disease detection model.
