# AI Models

This folder contains the trained AI models used in MediVision AI.

---

# Included Models

## oral_model.h5

Description:
TensorFlow-based oral disease classification model trained using oral disease datasets.

Detects:
- Caries
- Gingivitis
- Mouth Ulcer
- Tooth Discoloration

---

## skin_model.h5

Description:
MobileNetV2 transfer learning based skin disease classification model.

Detects multiple skin diseases including:
- Acne
- Psoriasis
- Vitiligo
- Eczema
- Warts
and more.

---

# Framework Used

- TensorFlow
- Keras
- MobileNetV2
