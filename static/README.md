# Static Files

This folder contains all static resources used in the MediVision AI web application.

---

# Folder Structure

static/

├── css/

├── images/

├── uploads/

└── profile_pics/

---

# css/

Contains all stylesheet files used for:
- UI Design
- Responsive Layout
- Dashboard Styling
- Login/Register Styling
- Animation Effects

---

# images/

Contains application images such as:
- Background images
- Icons
- Analysis section images
- UI graphics

---

# uploads/

Stores uploaded user images temporarily for:
- Skin disease analysis
- Oral disease analysis

These uploaded images are processed by the AI models for prediction.

---

# profile_pics/

Stores user profile images uploaded through the profile section.

Includes:
- Default profile image
- User uploaded profile pictures

---

# Purpose

The static folder is used to manage:
- Frontend assets
- User-uploaded images
- Application styling
- Visual resources
