# Templates

This folder contains all HTML templates used in the MediVision AI Flask application.

---

# Included Pages

## index.html
Splash screen / landing page.

## login.html
User login page.

## register.html
User registration page.

## verify_otp.html
OTP verification page for email authentication.

## dashboard.html
Main user dashboard.

## oral.html
Oral disease analysis page.

## skin.html
Skin disease analysis page.

## result.html
Displays AI prediction results.

## history.html
Displays previous prediction history.

## profile.html
User profile management page.

---

# Framework Used

- Flask Templating Engine
- HTML5
- CSS3
- Bootstrap
